# 猜词词库说明

## 数据结构

猜词词库位于：

```text
server/src/modules/guessword/data/words.json
```

每个目标词包含：

- `word`：目标答案
- `aliases`：强近义词，命中后给高分
- `related`：强关联词，命中后给中高分
- `hints`：弱提示词，命中后给中等分
- `category`：粗分类，用于同类词辅助评分
- `pinyin`：拼音字段，目前仅保留给后续扩展

## 当前评分层级

- 猜中目标词：`100`
- 命中强近义词：`92`
- 命中强关联词：`78`
- 命中弱提示词：`56`
- 未命中时，根据字符重合与同类词关系计算辅助分

## 低质量输入

以下输入会被判为低质量：

- 非中文或超过 8 个中文字符
- 3 个及以上完全相同字符堆叠，例如 `生生生`
- 4 个及以上字符且重复比例过高，例如 `医生生生`

低质量输入会返回 `0` 分，并带 `message` 提示。
